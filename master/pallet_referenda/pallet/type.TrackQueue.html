Expand description

The sorted list of referenda ready to be decided but not yet being decided, ordered by conviction-weighted approvals.

This should be empty if DecidingCount is less than TrackInfo::max_deciding.

Storage type is StorageMap with key type TrackIdOf < T, I > and value type BoundedVec < (ReferendumIndex, T :: Votes), T :: MaxQueued >.
