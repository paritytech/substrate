pub fn push_block_announce_validation( &mut self, who: PeerId, announce: BlockAnnounce<B::Header> )

Push a block announce validation.

It is required that ChainSync::poll_block_announce_validation is called later to check for finished validations. The result of the validation needs to be passed to SyncingEngine::process_block_announce_validation_result to finish the processing.

Note

This will internally create a future, but this future will not be registered in the task before being polled once. So, it is required to call ChainSync::poll_block_announce_validation to ensure that the future is registered properly and will wake up the task when being ready.
